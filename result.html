벨라소-비지네르 비닉 기법 해독¶

해독 과정 요약:

  1. 암호문 준비: 해독할 암호문을 계산하기 편한 형태로 가공합니다.
  2. 비닉용 건 찾기: 벨라소-비지네르 비닉 기법의 복호용 건을 찾습니다.
  3. 암호문 복호화: 복호화 함수를 사용하여 암호문을 복호화합니다.
  4. 복호화 결과 가공: 다자 환자식 비닉 기법에 맞춰 복호화된 메시지와 암호문의 각 글자가 각각 대응되어 한 눈에 보기 편하도록 가공합니다.

from collections import Counter

relative_frequencies = {}
alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
key_length = len(substrings)

# Calculate relative frequencies for each substring
for i, substring in enumerate(substrings):
    substring_name = f"Substring {i+1}"
    substring_length = len(substring)
    relative_frequencies[substring_name] = {}
    char_counts = Counter(substring) # Use Counter for efficient counting

    for char in alphabet:
        count = char_counts.get(char, 0) # Get count, default to 0 if char not present
        relative_frequencies[substring_name][char] = count / substring_length

    # Calculate and print the average of relative frequencies within this substring
    substring_freqs = relative_frequencies[substring_name] # Use the dictionary directly
    min_freq = min(substring_freqs.values())
    max_freq = max(substring_freqs.values())
    average_freq = sum(substring_freqs.values()) / len(alphabet)

    # Find the character(s) with min and max frequencies
    min_chars = [char for char, freq in substring_freqs.items() if freq == min_freq]
    max_chars = [char for char, freq in substring_freqs.items() if freq == max_freq]


    print(f"{substring_name} Relative Frequencies:")
    for char in alphabet:
        print(f"  {char}: {relative_frequencies[substring_name][char]:.6f}")
    print(f"Minimum Relative Frequency in {substring_name}: {min_freq:.6f} (Character(s): {', '.join(min_chars)})")
    print(f"Maximum Relative Frequency in {substring_name}: {max_freq:.6f} (Character(s): {', '.join(max_chars)})")
    print(f"Average Relative Frequency in {substring_name}: {average_freq:.6f}")
    print("-" * 30) # Separator for clarity

import matplotlib.pyplot as plt
from collections import Counter

def plot_relative_frequency(text):
    """
    Calculates and plots the relative frequency of letters in a given text.

    Args:
        text (str): The input text to analyze.
    """
    # Convert the text to lowercase and filter out non-alphabetic characters
    cleaned_text = ''.join(filter(str.isalpha, text.lower()))

    if not cleaned_text:
        print("Error: No alphabetic characters found in the text.")
        return

    # Count the frequency of each character
    char_counts = Counter(cleaned_text)

    # Calculate the total number of characters
    total_chars = len(cleaned_text)

    # Calculate the relative frequency of each character
    relative_frequencies = {char: count / total_chars for char, count in char_counts.items()}

    # Sort the frequencies alphabetically for consistent plotting
    sorted_freqs = sorted(relative_frequencies.items())
    chars, freqs = zip(*sorted_freqs)

    # Create the bar chart
    plt.figure(figsize=(12, 6))
    plt.bar(chars, freqs, color='skyblue')

    # Add labels and a title
    plt.title('Relative Frequency of Characters in a Text')
    plt.xlabel('Characters')
    plt.ylabel('Relative Frequency')
    plt.grid(axis='y', linestyle='--', alpha=0.7)

    # Display the plot
    plt.show()

# --- Example Usage ---

# You can replace this with your own text or a passage from a book.
sample_text = """
The quick brown fox jumps over the lazy dog.
This is an example text to demonstrate character frequency analysis.
Cryptanalysis often begins with frequency analysis to find patterns.
"""

# Call the function with the sample text
plot_relative_frequency(sample_text)

2.6. Chi-squared 테스트를 통한 비닉용 건 추정¶

각 Substring의 빈도 분포와 영어 평문의 빈도 분포를 비교하는 Chi-squared 테스트를 사용하여 각 Substring에 적용된 시프트 값을 추정하고, 이를 통해 최종 암호용 건을 추정합니다.

코드 설명: Chi-squared 테스트를 통한 키 추정¶

이 코드는 이전 단계에서 추정된 키 길이를 사용하여 각 substring에 대한 Chi-squared 테스트를 수행하고, 이를 통해 벨라소-비지네르 비닉 기법의 각 비닉용 건 문자를 추정합니다.

주요 기능:

  1. 영어 문자 빈도 설정: 일반적인 영어 텍스트의 알파벳 문자 상대 빈도 데이터를 english_freq 딕셔너리에 저장합니다. 이 데이터는 Chi-squared 테스트의 '기대값'으로 사용됩니다.
  2. Chi-squared 테스트 수행:

    • 추정된 key_length를 기반으로 각 substring에 대해 반복합니다.

    • 각 substring에 대해 0부터 25까지 가능한 모든 시프트 값(shift)을 적용하여 가상으로 시프트된 문자열을 생성합니다.

    • 시프트된 문자열의 실제 문자 빈도(observed_counts)를 계산합니다.

    • 각 문자에 대해 Chi-squared 통계량을 계산합니다. 이 통계량은 관찰된 빈도와 영어 평문의 기대 빈도 간의 차이를 측정합니다.

      공식은 ((관찰된 빈도 - 기대 빈도) ** 2) / 기대 빈도입니다.

    • 가장 낮은 Chi-squared 값을 갖는 시프트 값을 해당 substring의 최적 시프트로 결정합니다. Chi-squared 값이 낮을수록 해당 시프트가 적용된 문자열의 빈도 분포가 영어 평문의 빈도 분포와 유사하다는 의미입니다.

  3. 키 문자 추정: 각 substring의 최적 시프트 값을 바탕으로 해당하는 Vigenère 키 문자를 추정합니다. Chi-squared 테스트에서 찾은 최적 시프트는 암호문 문자에 '더했을 때' 평문과 유사한 빈도를 보이는 시프트입니다. 벨라소-비지네르 복호화는 '빼기' 연산이므로, 실제 비닉용 건 문자는 이 시프트의 역에 해당합니다 (26 - shift) mod 26.
  4. 추정 비닉용 건 생성: 추정된 각 키 문자를 합쳐서 최종적인 잠재적 암호용 건을 생성합니다.
  5. 결과 출력: 각 substring에 대한 최적 시프트와 그 때의 Chi-squared 값, 그리고 최종적으로 추정된 잠재적 키를 출력합니다.

이 코드는 통계적 방법을 사용하여 암호 분석의 핵심 단계인 비닉용 건 추정을 자동화합니다.

import math

english_freq = {

    'G': 2.015, 'H': 6.094, 'I': 6.966, 'J': 0.153, 'K': 0.772, 'L': 4.025,
    'M': 2.406, 'N': 6.749, 'O': 7.507, 'P': 1.929, 'Q': 0.095, 'R': 5.987,
    'S': 6.327, 'T': 9.056, 'U': 2.758, 'V': 0.978, 'W': 2.360, 'X': 0.150,
    'Y': 1.974, 'Z': 0.074
}

alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
key_length = len(substrings) # Use the actual key length from the substrings

estimated_shifts = {}

print(f"Performing Chi-squared test for each substring (key length {key_length}):")

for i in range(key_length):
    substring = substrings[i]
    substring_name = f"Substring {i+1}"
    min_chi_squared = float('inf')
    best_shift = 0

    for shift in range(26):
        shifted_substring = ""
        for char in substring:
            if char in alphabet:
                char_index = alphabet.index(char)
                shifted_index = (char_index + shift) % 26 # Apply shift
                shifted_substring += alphabet[shifted_index]
            else:
                shifted_substring += char # Keep non-alphabetic chars

        # Calculate observed frequencies for the shifted substring
        observed_counts = Counter(shifted_substring)
        substring_length = len(shifted_substring)

        # Calculate Chi-squared statistic
        chi_squared = 0
        for char in alphabet:
            observed_freq = observed_counts.get(char, 0)
            expected_freq = (english_freq.get(char, 0) / 100) * substring_length # Convert percentage to count
            # Avoid division by zero for expected_freq
            if expected_freq > 0:
                chi_squared += ((observed_freq - expected_freq) ** 2) / expected_freq
            # If expected_freq is 0 but observed is not, this contributes significantly
            elif observed_freq > 0:
                 chi_squared += observed_freq ** 2 / 0.01 # Add a small penalty, avoid inf

        # Update best shift if the current Chi-squared is lower
        if chi_squared < min_chi_squared:
            min_chi_squared = chi_squared
            best_shift = shift

    estimated_shifts[substring_name] = best_shift
    print(f"  {substring_name}: Best shift is {best_shift} (Chi-squared: {min_chi_squared:.2f})")


# Construct the potential key from the estimated shifts
# The shift found by the Chi-squared test is the amount we added to the ciphertext
# to make its frequency distribution look like English plaintext.
# To decrypt, we subtract the key. If C - K = P, then K = C - P.
# If shifting C by 's' gives P (C + s = P mod 26), then s = (P - C) mod 26.
# The shift 's' found here is the amount to ADD to the ciphertext character to get plaintext.
# The key character 'K' is the one such that C - K = P.
# So, K = (C - P) mod 26.
# If C + s = P, then s = P - C.
# K = -(P - C) = C - P.
# So K = -s mod 26, or (26 - s) mod 26.

potential_key_chars = [alphabet[(26 - shift) % 26] for shift in estimated_shifts.values()]
potential_key = "".join(potential_key_chars)

print(f"\nPotential key based on Chi-squared test: {potential_key}")

def vigenere_decrypt(encrypted_text, key):
    decrypted_text = ""
    alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    key = key.upper()
    key_length = len(key)

    for i, char in enumerate(encrypted_text.upper()):
        if char in alphabet:
            char_index = alphabet.index(char)
            key_char = key[i % key_length]
            key_index = alphabet.index(key_char)

            # Vigenere decryption formula: (C - K) mod 26
            decrypted_index = (char_index - key_index) % 26
            decrypted_char = alphabet[decrypted_index]
            decrypted_text += decrypted_char
        else:
            decrypted_text += char  # Handle non-alphabetic characters

    return decrypted_text

def plot_relative_frequency(text, ax, title, color='skyblue'):
    """
    Calculates and plots the relative frequency of letters in a given text on a specified axes.

    Args:
        text (str): The input text to analyze.
        ax (matplotlib.axes.Axes): The axes object to plot on.
        title (str): The title for the plot.
        color (str): The color of the bars.
    """
    cleaned_text = ''.join(filter(str.isalpha, text.lower()))
    if not cleaned_text:
        print(f"Error: No alphabetic characters found in the text for {title}.")
        return

    char_counts = Counter(cleaned_text)
    total_chars = len(cleaned_text)
    relative_frequencies = {char: count / total_chars for char, count in char_counts.items()}

    sorted_freqs = sorted(relative_frequencies.items())
    chars, freqs = zip(*sorted_freqs)

    ax.bar(chars, freqs, color=color)
    ax.set_title(title)
    ax.set_xlabel('Characters')
    ax.set_ylabel('Relative Frequency')
    ax.grid(axis='y', linestyle='--', alpha=0.7)
